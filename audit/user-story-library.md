# User Story Library — interieur.expert

Datum: 12 maart 2026
Status: Definitief

---

## Structuur per story

Elke story bevat:
- **Story ID** — unieke identificatie
- **Persona** — wie, leeftijd, context
- **Doel** — wat de gebruiker wil bereiken
- **Instappunt** — hoe de gebruiker op de site terechtkomt
- **Hoofdpad** — de verwachte stappen
- **Slagingscriteria** — wanneer is het bezoek succesvol?
- **Frustratietriggers** — wat kan misgaan?
- **Bedrijfsrelevantie** — waarom dit scenario belangrijk is
